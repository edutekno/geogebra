3.1 Pendahuluan

Bagian 3 ini akan membahas beberapa tool-tool GeoGebra yaitu Perpendicular line ( garis tegak lurus), Paralllel Line (garis sejajar), Angle bisector (garis pembagi sudut ) dan Tangent (garis singgung). Untuk memulai pembelajaran, silahkan menyimak video pengantar berikut ini. Setelah itu silahkan melanjutkan dengan mengerjakan penugasan yang diberikan.

Prasyarat untuk mengikuti aktivitas ini adalah Anda harus sudah menguasai tool GeoGebra terkait titik, garis, poligon dan lingkaran

3.7 Rangkuman dan Tindak Lanjut

Pada tahap ini Anda telah memelajari dan menguasai beberapa tool dan fitur GeoGebra, khususnya mengenai tool Perpendicular line ( garis tegak lurus), Paralllel Line (garis sejajar), Angle bisector (garis pembagi sudut ) dan Tangent (garis sisnggung) Beberapa tool ini dapat digunakan untuk mengembangkan media pembelajaran untuk beberapa topik matematika.
Untuk eksplorasi lebih lanjut, Anda dapat mengikuti tautan berikut ini untuk mendapatkan contoh dan ide pada beberapa topik terkait: Investigasi gradien (slope) pada garis sejajar, Investigasi gradien (slope) pada garis tegak lurus.
